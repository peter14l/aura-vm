Execute AI Code in 5 Milliseconds.

A hyper-fast, deeply isolated WebAssembly MicroVM built for AI Agent startups. Stop relying on slow Docker containers. Secure your host servers and execute untrusted code instantly.

Start Building Securely cargo add aura_vm

⛽ Instruction Fuel Limits

Prevents AI agents from locking up your CPU with infinite loops. Aura-VM force-kills the sandbox if the execution exceeds your strict fuel constraints.

🛡️ Memory Sandboxing

Strict OOM (Out Of Memory) protection. Cap the WebAssembly memory footprint to exactly 10MB per instance to protect your host servers.

⚡ Millisecond Cold Starts

Built purely in Rust using the Wasmtime engine. Boots, executes, and drops the isolated state environment in sub-5 milliseconds.
